## 🎮 游戏内道具应用流程

```
用户购买道具 → 存入背包 → 装备道具 → 游戏读取装备 → 应用效果
     │              │           │              │           │
     ▼              ▼           ▼              ▼           ▼
   支付成功      localStorage   标记已装备    游戏初始化    渲染时
   发放道具      存储道具数据   保存到 equipped 读取装备    应用样式
```

### 数据存储结构

```javascript
// 用户背包数据
userItems: [
  { id: "skin_gold", type: "skin", name: "黄金皮肤", boughtAt: "2024-03-05" },
  { id: "effect_fire", type: "effect", name: "火焰特效", boughtAt: "2024-03-05" }
]

// 已装备道具
equippedItems: {
  skin: "skin_gold",      // 当前装备的皮肤
  effect: "effect_fire",  // 当前装备的特效
  cardback: "back_dragon", // 当前装备的卡背
  frame: "frame_king"     // 当前装备的头像框
}
```

## 🔧 技术实现细节

### 皮肤应用代码
```javascript
// 游戏加载时读取装备的皮肤
const equippedSkin = localStorage.getItem('equipped_skin') || 'default';

// 渲染手牌时应用皮肤
function renderHandCard(card, skin) {
  const cardElement = document.createElement('div');
  cardElement.className = `hand-card skin-${skin}`;
  // 根据皮肤类型应用不同CSS样式
}
```

### 特效触发代码
```javascript
// 出牌时触发特效
function playCard() {
  const equippedEffect = localStorage.getItem('equipped_effect');
  
  // 显示特效
  if(equippedEffect === 'effect_fire') {
    showFireEffect();
  } else if(equippedEffect === 'effect_ice') {
    showIceEffect();
  }
  
  // 继续出牌逻辑
  processCardPlay();
}
```
