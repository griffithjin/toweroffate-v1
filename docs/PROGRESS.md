## 🎯 核心逻辑确认

### 牌组构成
- 主游戏: 208张 (4副标准扑克)
- 连胜模式: 104张 (2副标准扑克)

### 守卫机制
- 13名守卫
- 每名守卫: 13张守卫牌 + 3张激怒牌
- 激怒牌明牌展示

### 激怒牌规则
| 匹配类型 | 效果 |
|---------|------|
| 点数一致 或 花色一致 | 回退1层 |
| 点数花色完全一致 | 回退2层 |
| 出发层玩家 | 不受影响 |

### 首登者机制
- 首位登顶13层成为首登者
- 控制最高失守层守卫
- 可主动使用激怒牌
