# 命运塔·首登者 - 新版完整游戏逻辑

## 核心规则（2026-03-06 最新版）

### 牌组系统
- **总牌数**: 208张（4副标准扑克牌）
- **守卫牌**: 13名守卫各抽13张 = 169张（隐藏，不明示）
- **激怒牌**: 从剩余39张中每位守卫抽3张 = 39张（明牌展示）

### 玩家手牌
- 每位玩家拥有**完整52张牌**
- 每轮出1张牌（出完从手牌移除）
- 手牌耗尽仍未到13层 = 失败

### 激怒牌机制（关键）
- **3层/6层/9层守卫**: 玩家超过该层时触发
- **其他守卫**: 玩家距离3层内时随机触发
- **触发效果**:
  - 玩家出牌与激怒牌**花色或点数一致** → 回退1层
  - 玩家出牌与激怒牌**完全一致** → 回退2层
  - 最多退到出发层（出发层玩家不受影响）

### 守卫亮牌规则
- 玩家出牌后，守卫亮出1张牌（从13张中选）
- 守卫13张牌 + 3张激怒牌全部用完 = 守卫失守
- 失守层变为出发层

### 首登者规则
- 首位到达13层 = 首登者
- 首登者变成最高失守层守卫
- 重新激活守卫权限（13张牌+3张激怒牌）
- 可策略性发动激怒牌

### 团队模式
- 队友登顶13层 → 成为当前最高失守层守卫
- 无私守层时 → 变为-1层守卫
- 可选择赠送1/3剩余手牌给队友
- 一队全员登顶 = 胜利

### 积分系统
- 每局1000分
- 胜利者: 200基础分 + 800贡献分
- 失败者: -200基础分 - 800分（按表现扣）
- 贡献计算: 闯层成功率、队友贡献、守层贡献

---

## 连胜模式（简化版）
- 104张牌（2副）
- 守卫每层4张牌（可重复）
- 个人闯关，守卫牌亮明失效
- 失败不扣分，3层后每层加分
- 公式: 50 + (层数-3)*10
