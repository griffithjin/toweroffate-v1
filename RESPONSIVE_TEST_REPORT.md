## 🔧 技术实现详情

### CSS变量系统

```css
:root {
  /* 颜色 */
  --color-primary: #FFD700;
  --color-bg-dark: #0a0a1a;
  
  /* 响应式尺寸 */
  --max-width: 100%; /* 手机 */
  --btn-height: 48px;
  --font-base: 14px;
  
  /* 平板覆盖 */
  @media (min-width: 768px) {
    --max-width: 720px;
    --btn-height: 52px;
    --font-base: 16px;
  }
  
  /* 电脑覆盖 */
  @media (min-width: 1024px) {
    --max-width: 1200px;
    --btn-height: 56px;
    --font-base: 16px;
  }
}
```

### 响应式工具类

- `.container` - 响应式容器
- `.grid` / `.flex` - 布局系统
- `.hide-mobile` / `.show-desktop` - 显示控制
- `.btn` / `.btn-full` - 按钮系统
- `.card` / `.card-grid` - 卡片系统
- `.nav-mobile` / `.nav-desktop` - 导航系统
